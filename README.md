# ngamuk-beli-server

e commerce server

Tech :

- NodeJs
- Sequelize
- JWT
- Bcrypt
- Postgres
- Dotenv

To run this server

- create file .env and set config reference at .env.example
- npx sequelize db:create
- npx sequelize db:migrate
- npx sequelize db:seed:all
- npm run start
